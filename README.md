## Usage

To utilize `Cub3d` follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/redadoo/Cub3d.git
```

2. enter Cub3d Folder
```bash
cd Cub3d/
```

3. compile
```bash
make 
```
4. play
```bash
make play 
```
